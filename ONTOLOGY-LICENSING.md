# DISO Ontology Member Licence Records

Every member ontology (i.e., an ontology in the DISO collection) had its upstream license fetched (verbatim) from its authoritative source. This is recorded in a `LICENSE` file local to each ontology's home directory. Each local LICENSE file contains the upstream licence text, included verbatim, between RFC 4880 style markers with a SHA-256 checksum for verifiability.

## Licensing table

The Source(s) column links the origin of each verbatim licence text; `§n` matches the numbered verbatim section in the local `LICENSE` file. The *SPDX* column uses [SPDX licence expressions](https://spdx.org/licenses/); `LicenseRef-*` denotes terms with no SPDX identifier (e.g. MITRE terms of use, or `LicenseRef-author-permission` for a private written grant from the authors) and `NOASSERTION` means no licence could be established.

The **Author contacted / responded / granted** columns record direct outreach to rights holders, populated only from maintainer-held correspondence (dates in ISO form). Where the grant is by private correspondence it is **not** publicly reproduced; it is verifiable by contacting the ontology's original authors. `n/a` means the column does not apply (e.g. a public licence made contact unnecessary).
